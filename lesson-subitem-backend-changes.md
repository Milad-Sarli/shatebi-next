# مدیریت زیرشاخه‌های دروس

این سند قابلیت‌های جدیدی را برای مدیریت زیرشاخه‌های دروس در سیستم توضیح می‌دهد. این قابلیت‌ها به کاربر اجازه می‌دهند تا دروس را به عنوان زیرشاخه به یک درس والد اضافه یا از آن جدا کند و همچنین لیست دروس موجود برای انتساب به عنوان والد را مشاهده کند.

## API‌های پیاده‌سازی شده

### 1. حذف والد از یک درس (جدا کردن زیرشاخه)
این API به شما امکان می‌دهد تا یک درس را از والد خود جدا کنید. با این کار، `parent_id` درس به `null` تغییر کرده و درس به یک درس مستقل تبدیل می‌شود.

**مسیر:** `PUT /api/droos/{id}/remove-parent`

**پارامترها:**
- `id`: شناسه درسی که می‌خواهید از والد آن جدا شود.

**پاسخ موفق:**
```json
{
  "status": true,
  "message": "درس با موفقیت از والد جدا شد",
  "data": {
    "id": 123,
    "title": "عنوان درس",
    "parent_id": null,
    // سایر اطلاعات درس
  }
}
```

### 2. اضافه کردن زیرشاخه به یک درس
این API به شما امکان می‌دهد تا یک درس موجود را به عنوان زیرشاخه به یک درس والد اضافه کنید. `parent_id` درس انتخاب شده به شناسه درس والد تغییر می‌کند.

**مسیر:** `PUT /api/droos/{parentId}/add-child`

**پارامترها:**
- `parentId`: شناسه درس والد.
- Body:
```json
{
  "child_id": 456
}
```

**پاسخ موفق:**
```json
{
  "status": true,
  "message": "درس با موفقیت به عنوان زیرشاخه اضافه شد",
  "data": {
    "id": 456,
    "title": "عنوان زیرشاخه",
    "parent_id": 123,
    // سایر اطلاعات درس
  }
}
```

### 3. دریافت لیست دروس قابل انتساب به عنوان والد
این API لیستی از دروس را برمی‌گرداند که می‌توانند به عنوان والد به دروس دیگر اختصاص داده شوند (یعنی دروسی که خودشان والد ندارند).

**مسیر:** `GET /api/droos/available-for-parent`

**پارامترها:**
- `exclude_id`: (اختیاری) شناسه درسی که می‌خواهید از لیست نتایج حذف شود.

**پاسخ موفق:**
```json
{
  "status": true,
  "data": [
    {
      "id": 789,
      "title": "عنوان درس 1",
      "parent_id": null,
      // سایر اطلاعات درس
    },
    {
      "id": 790,
      "title": "عنوان درس 2",
      "parent_id": null,
      // سایر اطلاعات درس
    }
    // ...
  ]
}
```
