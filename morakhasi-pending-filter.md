# API فیلتر مرخصی‌های در انتظار تایید

## نقطه پایانی (Endpoint)
```
GET /api/morakhasi/pending-acceptance
```

## توضیحات
این API برای دریافت لیست مرخصی‌های در انتظار تایید استفاده می‌شود و امکان فیلتر کردن بر اساس نوع مرخصی را فراهم می‌کند.

## پارامترهای Query

### فیلتر نوع مرخصی (type)
- **پارامتر**: `type`
- **نوع**: integer
- **اختیاری**: بله
- **مقادیر مجاز**:
  - `1`: مرخصی ساعتی
  - `3`: مرخصی چند روزه
  
**نکته**: نوع `2` در این API پشتیبانی نمی‌شود.

### سایر پارامترها
- **search**: جستجو در نام، دلیل و پیام ولی (اختیاری)
- **per_page**: تعداد آیتم در هر صفحه (پیش‌فرض: 10)

## نمونه درخواست‌ها

### دریافت تمام مرخصی‌های در انتظار تایید
```http
GET /api/morakhasi/pending-acceptance
```

### فیلتر مرخصی‌های ساعتی
```http
GET /api/morakhasi/pending-acceptance?type=1
```

### فیلتر مرخصی‌های چند روزه
```http
GET /api/morakhasi/pending-acceptance?type=3
```

### ترکیب فیلتر نوع با جستجو
```http
GET /api/morakhasi/pending-acceptance?type=1&search=احمد
```

### تنظیم تعداد آیتم در صفحه
```http
GET /api/morakhasi/pending-acceptance?type=3&per_page=20
```

## نکات مهم

1. **دسترسی**: کاربران غیر ادمین فقط مرخصی‌های خود را مشاهده می‌کنند
2. **وضعیت**: فقط مرخصی‌های با وضعیت `2` (در انتظار تایید) نمایش داده می‌شوند
3. **زمان**: فقط مرخصی‌هایی که هنوز منقضی نشده‌اند نمایش داده می‌شوند
4. **فیلتر نوع**: فقط نوع‌های `1` و `3` پذیرفته می‌شوند، سایر مقادیر نادیده گرفته می‌شوند

## کدهای خطا

- **401**: عدم احراز هویت
- **403**: عدم دسترسی
- **422**: پارامترهای نامعتبر
